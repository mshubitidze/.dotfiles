# Setup

## install command line tools

```shell
xcode-select --install
```

## Homebrew

```shell
brew bundle --file ~/.dotfiles/Brewfile
```

## create symlinks to config files

```shell
ln -s ~/.dotfiles/.gitconfig ~/.gitconfig
ln -s ~/.dotfiles/.vimrc ~/.vimrc
ln -s ~/.dotfiles/.zshrc ~/.zshrc
```
