## Setup

### git clone this repo in ~/
1. install cli tools
2. install [**Brew**](https://brew.sh), git
3. install alacritty via brew
4. install [**Oh My Zsh**](https://ohmyz.sh/) / [**Starship**](https://starship.rs/)
5. install install [**Brewfile**](https://github.com/mshubitidze/.dotfiles/blob/main/Brewfile)
6. install neovim/[**NvChad**](https://nvchad.com/)/[**Lunarvim**](https://www.lunarvim.org/)
7. install [**Klack**](https://tryklack.com), [**Quickshade**](https://apps.apple.com/us/app/quickshade/id931571202)

### symlink dots
1. ln -s ~/.dotfiles/nvim/custom ~/.config/nvim/lua/custom
2. ln -s ~/.dotfiles/alacritty ~/.config/alacritty
3. ln -s ~/.dotfiles/yabai ~/.config/yabai
4. ln -s ~/.dotfiles/skhd ~/.config/skhd
